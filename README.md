# E-commerce Global Sales Project: Multi-Dimensional Performance Analysis

## Project Overview  
This project delivers a comprehensive analysis of global e-commerce sales performance across regions, product categories, devices, and time. Using SQL, Python, and Tableau, it identifies key performance drivers, customer behavior patterns, and market opportunities to inform strategic decision-making.

## Objectives  
- Integrate and clean large-scale sales data using SQL and Python  
- Uncover geographic, demographic, and behavioral trends  
- Apply statistical testing to validate hypotheses and detect significant patterns  
- Visualize insights through interactive dashboards in Tableau and executive summaries

## Methodology  
- **ETL Process**:  
  - **Extract**: SQL queries via BigQuery API to retrieve raw sales, traffic, and user data  
  - **Transform**: Data cleaning, normalization, and enrichment using Python (Pandas, NumPy)  
  - **Statistical Techniques**:  
    - Descriptive statistics & EDA  
    - Correlation analysis with significance testing  
    - Hypothesis testing (T-tests, Chi-square, Shapiro-Wilk, Mann-Whitney U, Bayesian Probability, Bootstrap)  
    - Time series analysis for seasonality detection
  - **Load**: Structured datasets prepared for analysis and visualization in Colab and Tableau
